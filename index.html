🛡️ Politique de confidentialité & RGPD

📋 Données traitées

Cet outil traite des données professionnelles : noms de classes, noms de professeurs, horaires et plannings. Ces données peuvent constituer des données à caractère personnel au sens du RGPD (Règlement UE 2016/679).

✅ Ce que cet outil garantit

✓ Aucun serveur externe — toutes les données restent dans votre navigateur, sur votre appareil uniquement.
✓ Aucun cookie de tracking — pas de publicité, pas d'analytics, pas de profiling.
✓ Stockage local uniquement — les sessions sauvegardées utilisent le localStorage de votre navigateur. Elles ne sont accessibles qu'à vous.
✓ Aucune transmission à des tiers — les données ne sont ni vendues, ni partagées, ni exploitées commercialement.
✓ Fonctionnement hors ligne — le site peut fonctionner sans connexion internet une fois chargé.

⚠️ Points d'attention

⚠️ Bibliothèques JavaScript — pour fonctionner, ce site charge des bibliothèques depuis des serveurs CDN externes (Cloudflare). Votre adresse IP peut être transmise lors du chargement. Pour un usage 100% autonome, téléchargez les fichiers JS en local.
⚠️ Responsabilité du responsable de traitement — l'établissement qui utilise cet outil est responsable du traitement des données saisies. Il lui appartient d'informer les personnes concernées (professeurs) conformément à l'article 13 du RGPD.

🏫 Base légale

L'utilisation de cet outil dans le cadre de la gestion des conseils de classe relève de l'exécution d'une mission d'intérêt public (article 6.1.e du RGPD), dans le cadre des missions de l'Éducation Nationale.

🗑️ Suppression des données

Pour effacer toutes les données stockées sur votre appareil : videz le localStorage de votre navigateur pour ce site, ou utilisez la navigation privée. Les sessions peuvent être supprimées individuellement depuis le panneau 🗂️ Sessions.
